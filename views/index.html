Burn Down Chart:

Sprint 2 Completed Stories:

  • Create a page displaying all events (3 points)

During the week, Nikhil and Aly worked on developing the code for the Sprint 2 user stories, while Shikha and Vinay took on the responsibility of testing the functionality and pushing the code live to the Heroku web application.

Following the completion of the tester’s work, the product owner carried out a final review of the sprint tasks and backlog to ensure accuracy.
